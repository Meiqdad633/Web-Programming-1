\# Temperature Converter



\## Project Description



This project is a simple web-based Temperature Converter developed using HTML, CSS, and JavaScript. The application allows users to convert temperatures between Celsius and Fahrenheit. It provides a user-friendly interface where users can enter a temperature, choose the conversion type from a drop-down menu, and click the \*\*Convert\*\* button to display the result.



The project demonstrates the use of JavaScript variables, functions, event listeners, control structures, and DOM manipulation to create an interactive web application.



\---



\## Features



\- Convert Celsius to Fahrenheit

\- Convert Fahrenheit to Celsius

\- Input validation for numeric values

\- Error message for invalid input

\- Clean and responsive user interface

\- Instant conversion when the Convert button is clicked

\## How to Run the Project



1\. Download the project files.

2\. Open \*\*TemperatureConverter.html\*\* in any modern web browser (Google Chrome, Microsoft Edge, Firefox, or Safari).

3\. Enter a temperature value.

4\. Select the desired conversion:

&#x20;  - Celsius to Fahrenheit

&#x20;  - Fahrenheit to Celsius

5\. Click the \*\*Convert\*\* button.

6\. View the converted temperature or an error message if the input is invalid.



\---



\## Temperature Conversion Formulas



\*\*Celsius to Fahrenheit\*\*



```

F = (C × 9/5) + 32

```



\*\*Fahrenheit to Celsius\*\*



```

C = (F − 32) × 5/9

```
