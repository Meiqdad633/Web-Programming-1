\# 📖 Introduction



This project is a simple \*\*Travel Blog Website\*\* created using HTML and CSS.  

It demonstrates how to structure web pages and apply styling to make them visually appealing and user-friendly.



The website consists of two main pages:



\- \*\*Homepage (home.html)\*\* → Interactive number blocks with nested travel destinations  

\- \*\*Travel Stories (stories.html)\*\* → Table-based layout with images and linked sections  



The goal of this assignment is to practice:

\- HTML page structure

\- CSS styling techniques

\- Internal navigation using links

\- Tables, lists, and images in web design



\---



\# 🏠 Homepage (home.html)



The homepage includes four interactive number blocks:



\- 1 → Europe  

\- 2 → Asia  

\- 3 → America  

\- 4 → Africa  



When a user clicks on any number, the page scrolls to a specific section showing:



\- Ordered lists of countries  

\- Nested unordered lists of cities  



\### ✨ Features:

\- Clickable number blocks

\- Colorful gradient design

\- Nested lists (countries → cities)

\- Card-style sections

\- Hover effects on buttons



\---



\# ✈️ Travel Stories Page (stories.html)



This page contains a travel table with three destinations:



\- Paris 🇫🇷 (Eiffel Tower)

\- Rome 🇮🇹 (Colosseum)

\- New York 🇺🇸 (Statue of Liberty)



Each image is clickable and jumps to a detailed section below the table.



\### ✨ Features:

\- HTML table layout

\- Image linking to sections

\- Alternating row colors

\- Hover effects on images

\- Smooth card-style sections



\---



\# 🎨 CSS Styling Summary



CSS was used to improve design and user experience.



\### Homepage Styling:

\- Gradient number blocks

\- Hover animations (lift effect)

\- Card-style sections with shadows

\- Clean typography and spacing



\### Travel Stories Styling:

\- Styled table with borders and padding

\- Alternating row colors

\- Image hover effects (opacity + zoom)

\- Rounded corners for modern look



According to MDN Web Docs (2024), CSS controls how HTML elements are displayed, including layout, colors, spacing, and typography, which is essential for creating visually appealing websites.
